毎月の更新手順

  1. Excelファイル（ソフトウェア部門_売上分析_2024-2026.xlsx）の 月次入力シート に当月データを入力
  2. 「📁 Excelで更新」ボタンでファイルをアップロード → グラフが即時更新
  3. 内容を確認したら「☁️ GitHubに保存」→ PAT 入力 → Push
  4. 1〜2分で GitHub Pages に反映される

PAT（Personal Access Token）の取得方法

  1. GitHub にログイン → 右上のアイコン → Settings
  2. 左メニュー最下部 → Developer settings → Personal access tokens → Tokens (classic)
  3. Generate new token → repo にチェック → Generate
  4. 発行されたトークンをコピー（一度しか表示されません）
  5. 「☁️ GitHubに保存」の PAT 欄に貼り付け（ブラウザが記憶）
